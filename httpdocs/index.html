This site contains listings and POI data from Wikivoyage - the free worldwide travel guide that anyone can edit. Listing are provided in different formats, ready to be used in another project. These formats include:
  • CSV
  • GPX
  • KML
  • SQL script to be used with SQLite
  • Raw OsmAnd XML
  • OsmAnd OBF file, ready to be used with OsmAnd
